Project 1: Classcube and Mergecube

Link to full project available on Github here.
Introduction: This first project allows students to become familiarized with the Unity environment and have them slowly begin working with importing and creating models. After cutting out two paper cubes and putting them together, with similarities between them, there are also a few differences that will take place. Both cubes will focus on having various elements on them, such as models, .png background images, C# scripts using OpenWeather API, and text fields. The main differences between the two will be regarding models, 8ball mystery sayings, and location for the use of the API.

How to use: When ready to run program, click the "Play" button at the top. Make sure to have a webcam connected. When running, simply point the webcam at one or both of the paper cubes. You will see your models come to life!
